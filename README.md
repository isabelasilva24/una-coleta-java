# 🌌 Sistema de Coleta de Dados - Seleção Interestelar

Este projeto é um sistema simples desenvolvido em **Java**, que realiza a coleta de dados de um candidato via terminal.

---

## 📌 Objetivo

O sistema simula um processo de seleção, solicitando informações do usuário e exibindo os dados coletados.

---

## 🛠️ Tecnologias Utilizadas

- Java ☕
- Scanner (entrada de dados via console)

---

## 🚀 Como executar o projeto

### 1. Compile o código:
```bash
javac ColetaDados.java
